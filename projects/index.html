VCCL

Variable controlled commandline language is an esoteric programming language that relies on variables for control flow and io. It's my first esolang and has a working interpreter writen in java. It has a fairly simple syntax and 11 total operations. Every command assigns a value to a variable which can be computed using an expression or got from an other variable.
